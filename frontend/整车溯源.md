功能概述
用户在项目选择框中选择"项目名-委托单号-样品编号"后，系统需计算并返回前25名贡献度数据。前端界面保持不变，仅后端接口逻辑调整。

数据查询规则
示例：选择 F123-1-1 项目时

筛选条件：项目名称 = F123，零件名称 ≠ 整车 SubstancesTestDetail表
数据处理：同一项目下相同零件名称的样品数据需求平均值

计算公式为
公式	实现方式
Qij = Cij / OTVj	在SubstancesTestDetail中
Oi = Σ Qij	
sum([d.qij for d in sample.voc_data.all() if d.qij])
ZOi = Σ Oi（同车型同零件不同阶段取平均）	
GOi = (Oi / ZOi) × 100%	Python 层计算即可
Wih = Cih / TVh	在SubstancesTestDetail中
Vi = Σ Wih	类似 Oi，对 wih 字段求和
ZVi = Σ Vi	对所有零部件的 Vi 求和
GVi = (Vi / ZVi) × 100%	同 GOi
