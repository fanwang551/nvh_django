# 需求优化文档

## 一、需求背景
优化 NTF 查询界面的热力图展示效果。

## 二、具体需求

### 需求1：热力图视觉优化 - 添加数据间隔
**目标**：提升热力图的可读性，使不同数据行之间有明确的视觉分隔

**详细说明**：
- 在每条数据对应的热力图行之间插入空白间隔行
- 空白行高度：固定值2px



**热力图效果草图**：

```

│ 测点A │████████████████████████████│ ← 数据行1

│ 测点B │████████████████████████████│ ← 数据行2
├───────┼────────────────────────────┤
│ 测点C │████████████████████████████│ ← 数据行3
└─────────────────────────────────────┘

优化后：
┌─────────────────────────────────────┐
│ 测点A │████████████████████████████│ ← 数据行1 
├───────┼────────────────────────────┤
│       │                            │ ← 空白行 
├───────┼────────────────────────────┤

├───────┼────────────────────────────┤
```
